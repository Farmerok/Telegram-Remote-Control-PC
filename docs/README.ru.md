## 📝 Введение

Добро пожаловать в **PCGuardControl**! Этот мощный инструмент позволяет удаленно управлять компьютером через простые команды в Telegram, делая вашу работу удобнее и эффективнее.

Скрипт поддерживает одного или нескольких администраторов. Например, на семейном компьютере вы можете предоставить доступ близким.

Мы регулярно выпускаем бесплатные обновления, чтобы поддерживать функциональность и безопасность скрипта.

## ⚙️ Настройка скрипта

Для настройки:

1. Запустите скрипт, чтобы создать файл `settings.ini` для ввода данных (пример ниже).
2. Или создайте файл `settings.ini`, скопируйте пример и замените данные на свои.

    ```ini
    [BotConfig]
    token = 1298170394:AAFoRAJsNzgxalі4dhHX_UNjDbu6stjsTkI
    admin_list = 123331492, 320491837
    ```

### Как получить данные

1. **Токен бота:**  
   - Найдите в Telegram [@BotFather](https://t.me/BotFather).  
   - Отправьте команду `/newbot` и следуйте инструкциям для создания бота.  
   - Получите токен, например: `123456789:ABCDefghIJKLMNOPQRSTUVWXYZ`.  
   - Сохраните его в `settings.ini`.

2. **ID администратора:**  
   - Найдите в Telegram `@userinfobot` или `@getmyid_bot`.  
   - Начните чат, чтобы получить ваш Telegram ID, например: `123456789`.  
   - Добавьте ID в `admin_list` в `settings.ini`. Для нескольких администраторов разделяйте ID запятыми (`,`).
