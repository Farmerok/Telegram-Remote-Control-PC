## 📝 Вступ

Ласкаво просимо до **PCGuardControl**! Цей потужний інструмент дозволяє віддалено керувати комп’ютером за допомогою простих команд у Telegram, роблячи вашу роботу зручнішою та ефективнішою.

Скрипт підтримує одного або кількох адміністраторів. Наприклад, на сімейному комп’ютері ви можете надати доступ своїм близьким.

Ми регулярно випускаємо безкоштовні оновлення для підтримки безпеки та функціональності скрипта.

## 📂 Керування файлами та папками

- **📁 Навігація по папках:** Швидко перемикайтеся між директоріями.
- **📂 Створення папок:** Створюйте нові папки будь-де.
- **✏️ Перейменування:** Змінюйте назви існуючих папок.
- **🔍 Сканування:** Переглядайте файли та папки в поточній директорії.
- **🔄 Зміна директорії:** Легко переходьте між папками.

---

## 🛠️ Додаткові функції

- **💻 Командний рядок:** Виконуйте команди в консолі Windows.
- **🚀 Запуск файлів:** Відкривайте файли з будь-якої директорії.
- **📥 Завантаження файлів:** Надсилайте файли на ПК.
- **📤 Скачування файлів:** Отримуйте файли з ПК у Telegram.
- **🗑️ Видалення:** Видаляйте файли чи папки за назвою.
- **🔗 Завантаження за посиланням:** Скачуйте файли за прямими посиланнями.
- **👮 Антикрадіжка:** Блокуйте екран при будь-якій активності.
- **✍️ Введення тексту:** Друкуйте текст на ПК через бот.
- **🗣 Голосові повідомлення:** Надсилайте голосові повідомлення, які відтворюватимуться на ПК.
- **📦 Буфер обміну:** Переглядайте та змінюйте вміст буфера.
- **👀 Моніторинг буфера:** Відстеження змін буфера обміну в реальному часі з повідомленнями.
- **🛡️ Брандмауер:** Увімкніть/вимкніть захист прямо в боті.
- **🖥 Керування монітором:** Увімкніть/вимкніть монітор.
- **⌨️ Блокування вводу:** Обмежуйте доступ до миші та клавіатури.
- **🪫 Контроль батареї:** Отримуйте сповіщення про низький заряд ноутбука.
- **🗂 Перезапуск провідника:** Перезапускайте провідник Windows.

## ⚠️ Важлива інформація

- Скрипт є власницьким і не має відкритого коду.
- Регулярні оновлення забезпечують безпеку та нові функції.
- **Що таке `update.exe`?**  
  Цей файл відповідає за автоматичне оновлення скрипта. Запустіть його, щоб завантажити останню версію без ручної установки.
- **⚠️ Не використовуйте один токен у кількох програмах одночасно** — лише в одному застосунку.
- **Про спрацювання антивірусів:**

  ```ini
  Деякі антивіруси можуть позначити скрипт як загрозу через його функцію віддаленого керування ПК.
  Це типова поведінка для подібних програм через їх функціонал.
  Скрипт абсолютно безпечний для використання.

  Вирішуйте самі, завантажувати його чи ні — це ваше право, і ми його поважаємо. Якщо ви довіряєте джерелу,
  додайте файл до виключень антивіруса, щоб уникнути хибних спрацьовувань.

  Скрипт надається безкоштовно, і ми підтримуватимемо його оновленнями для безпеки та функціональності.
  Завантаження безпечне, і ви отримаєте потужний інструмент для керування ПК через Telegram!
  ```

- **🚨 Відмова від відповідальності:**  
  Розробники не несуть відповідальності за незаконне використання скрипта. Не використовуйте його для дій, що порушують закон або права інших. Програму слід застосовувати лише в законних цілях.

## ⚙️ Налаштування скрипта

Для налаштування:

1. Запустіть скрипт, щоб створити файл `settings.ini` для введення даних (приклад нижче).
2. Або створіть файл `settings.ini`, скопіюйте приклад і замініть дані на свої.

    ```ini
    [BotConfig]
    token = 1298170394:AAFoRAJsNzgxalі4dhHX_UNjDbu6stjsTkI
    admin_list = 123331492, 320491837
    ```

### Як отримати дані

1. **Токен бота:**  
   - Знайдіть у Telegram [@BotFather](https://t.me/BotFather).  
   - Надішліть команду `/newbot` і дотримуйтесь інструкцій для створення бота.  
   - Отримайте токен, наприклад: `123456789:ABCDefghIJKLMNOPQRSTUVWXYZ`.  
   - Збережіть його в `settings.ini`.

2. **ID адміністратора:**  
   - Знайдіть у Telegram `@userinfobot` або `@getmyid_bot`.  
   - Почніть чат, щоб отримати ваш Telegram ID, наприклад: `123456789`.  
   - Додайте ID до `admin_list` у `settings.ini`. Для кількох адміністраторів розділяйте ID комами (`,`).
